Comandos Aprendidos

CSS Propiedades Principales

  1. background-color: Define el color de fondo de un elemento.
  2. color: Establece el color del texto.
  3. font-family: Especifica la fuente del texto.
  4. font-size: Define el tamaño del texto.
  5. margin: Agrega espacio fuera de un elemento, entre el borde y otros elementos.
  6. padding: Agrega espacio dentro de un elemento, entre su contenido y el borde.
  7. border: Define el borde de un elemento, incluyendo su grosor, estilo y color.
  8. display: Controla cómo se muestra un elemento (block, inline, flex, grid).
  9. position: Especifica cómo se posiciona un elemento (static, relative, absolute, fixed).
  10. z-index: Controla el orden de apilamiento de los elementos posicionados.
  11. opacity: Controla la transparencia de un elemento.
  12. box-shadow: Agrega sombra a un elemento para darle profundidad.
  13. border-radius: Redondea las esquinas de un elemento.
  14. flex: Define cómo se distribuyen los elementos dentro de un contenedor flexible.
  15. align-items: Alinea los elementos dentro de un contenedor flexible en el eje transversal.
  16. justify-content: Alinea los elementos dentro de un contenedor flexible en el eje principal.
  17. overflow: Controla cómo se maneja el contenido que desborda el área de un elemento.
  18. place-items: Combina align-items y justify-items en CSS Grid para alineación total.

JavaScript Métodos Avanzados

  1. addEventListener(): Registra un escuchador de eventos para detectar interacción del usuario (teclado, click).
  2. requestAnimationFrame(): Crea un bucle de animación sincronizado con el navegador para actualizar el juego constantemente.
  3. forEach(): Itera sobre arrays u objetos para ejecutar una función en cada elemento.
  4. Object.keys(): Extrae todas las propiedades de un objeto en forma de array.
  5. new Image(): Crea un objeto de imagen para cargar spritesheets desde archivos.
  6. getContext('2d'): Obtiene el contexto 2D del canvas para dibujar gráficos.
  7. drawImage(): Dibuja una imagen o sección de spritesheet en el canvas.
  8. getElementById(): Selecciona elementos HTML por su atributo id para manipularlos.
  9. classList.add/remove(): Agregan o quitan clases CSS del elemento para cambiar estilos dinámicamente.
  10. window (objeto global): Almacena variables y funciones accesibles desde cualquier módulo JavaScript.
  11. Closures: Funciones que capturan variables del scope externo para mantener estado privado.
  12. this (contexto): Referencia al objeto actual, esencial en métodos de clases como Player y Fire.

Lógica del Juego

El juego Space Survivor funciona mediante un bucle de actualización constante. El jugador controla una nave con las flechas del teclado y dispara con espacios. El sistema de enemigos genera naves enemigas que descienden aleatoriamente, y cada impacto suma puntos. El juego usa Canvas 2D para dibujar sprites desde spritesheets, detecta colisiones entre proyectiles y enemigos, y termina cuando se pierden todas las vidas. Cada módulo JavaScript (player.js, enemigos.js, shoot.js, vidas.js) maneja una responsabilidad específica del juego.
